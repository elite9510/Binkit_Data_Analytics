# Blinkit_Data_Analytics

## Blinkit Data Analytics is an end-to-end data analysis project designed to transform raw business data into meaningful insights. The project focuses on understanding customer behavior, sales trends, and operational performance using modern data analytics tools and techniques.

## Dataset
<a href="https://github.com/elite9510/Binkit_Data_Analytics/blob/main/BlinkIT%20Grocery%20Data.xlsx">Blinkit data</a>

## Process
Clean and preprocess sales and product datasets
Analyze outlet performance based on type, location, and size
Study customer buying patterns and product demand
Identify top-selling and low-performing items
Visualize KPIs using Power BI / Python
Provide actionable insights for business decisions
